Yoga Pose Classification and Feedback System

This project aims to develop a real-time Yoga Pose Classification and Feedback system using video-based joint angle analysis. It classifies nine yoga poses using a deep learning model trained on MediaPipe-extracted body landmarks and provides corrective feedback when joint angles deviate from ideal posture, enhancing at-home yoga practice safety and effectiveness.

Atmospheric Correction Using Deep Learning

This project leverages U-Net and R2U-Net deep learning architectures to enhance Sentinel-2 satellite imagery, significantly improving clarity, spatial detail, and processing efficiency. It outperforms traditional atmospheric correction techniques, with applications in environmental monitoring, agriculture, and land management. Tools used include PyTorch, TensorFlow, and advanced image processing methods.

FundChain Decentralized Crowdfunding Platform

FundChain is a decentralized crowdfunding platform built on Ethereum blockchain. It ensures transparency, security, and fraud prevention by recording all campaign activities on a public ledger. Users can create campaigns, contribute, and approve fund withdrawals through smart contracts, eliminating the risks associated with centralized systems.

Fraud Detection
